# 📝 To-Do + Daily Tracker Web App

A simple, clean, and fully functional **To-Do List + Daily Tracker** built using **HTML, Tailwind CSS, and JavaScript**.
This project focuses on usability, structured task management, and persistent storage using browser-based storage.

---

## 🚀 Features

### ✅ To-Do List (Page 1)

* Create **multiple sections (topics)**
* Add tasks inside each section
* ✔️ Mark tasks as complete
* 📅 Automatic date assignment (defaults to today)
* ✏️ Option to set custom dates
* 🗑️ Delete tasks and sections

---

### 🔵 Daily Tracker (Page 2)

* Create **daily sections** (e.g., Fitness, Study)
* Add repeatable daily tasks
* ✔️ Track completion using checkboxes
* 🗑️ Delete tasks and sections

---

### 💾 Data Persistence

* Uses **localStorage** (browser storage)
* Tasks are **automatically saved in real-time**
* Data remains after:

  * page refresh
  * closing and reopening browser

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **Tailwind CSS** – Styling
* **JavaScript (Vanilla)** – Logic & interactivity
* **localStorage API** – Data persistence

## ▶️ How to Run

1. Download or clone the repository
2. Open the folder in **VS Code**
3. Open `todo.html`

👉 Recommended:

* Install **Live Server extension**
* Right-click → **Open with Live Server**

---

## ⚠️ Notes

* Data is stored locally in your browser
* Clearing **site data/storage** will delete saved tasks
* Data is not synced across devices

---

## 🎯 Purpose of the Project

This project was built to:

* Practice **JavaScript DOM manipulation**
* Understand **client-side storage (localStorage)**
* Build a **real, usable application without a backend**
* Learn how to structure multi-page UI in a single HTML file
